官方职能包

每个职能包集成了人格定义 + 领域知识 + 可执行技能。所有职能包都继承 base-agent 基础层——提供浏览器自动化等通用能力。

创建与发布

将你的专业经验封装成可复用的职能包，分享给开发者社区。

1. 脚手架
agentbrief init
2. 自定义
编辑 personality.md，添加 markdown 文件到 knowledge/，添加目录到 skills/
3. 预览
agentbrief preview
4. 发布
推送到 GitHub → 向官方 registry 提交 PR
