The Difference

Same question, completely different answer.

Without a brief
You: "Review this code for issues"
Agent: "The code looks good overall. Consider adding some error handling and maybe a few more comments."
With security-auditor brief
You: "Review this code for issues"
Agent: "CWE-89 CRITICAL at line 23: SQL injection via string concatenation. Attack vector: attacker injects arbitrary SQL through userId. Fix: db.query('SELECT * FROM users WHERE id = $1', [userId])"

Create & Publish

Package your domain expertise into a reusable brief and share it with the developer community.

1. Scaffold
agentbrief init my-agent
2. Customize
Edit personality.md, add markdown files to knowledge/ and directories to skills/
3. Preview
agentbrief preview ./my-agent
4. Publish
Push your repo to GitHub → submit a PR to the official registry
