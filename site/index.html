AgentBrief

Pluggable role definitions for AI coding agents.

$ npm install -g agentbrief
$ agentbrief use security-auditor
# Your AI agent is now an OWASP security auditor

$ agentbrief search
# Browse 14 official briefs across dev, product, growth & ops

How It Works

Three steps. Under 60 seconds.

1

Install

npm install -g agentbrief

2

Apply a Brief

agentbrief use security-auditor

Injects role, knowledge, and rules into your agent's instruction files.

3

Start Coding

Open a new conversation. Your agent now behaves as a specialist.
